# CivicChain

Verify, pay, and build trust on the blockchain.

CivicChain is a decentralized civic payment and verification platform built on the Cardano blockchain and powered by Atala PRISM decentralized identities.

Live application: [https://civicchain-topaz.vercel.app/](https://civicchain-topaz.vercel.app/)

## About

CivicChain enables users to create and manage verifiable decentralized identities, carry out secure blockchain-based payments, and access transparent, tamper-proof verification records.

The platform is designed to support civic, financial, and identity-driven interactions where trust, transparency, and decentralization are essential.



## Features

* Decentralized digital identity creation and management using Atala PRISM
* Secure payments on the Cardano blockchain
* Transparent and immutable verification records
* User authentication and onboarding flows



## Technology Stack

* Frontend: Next.js and React
* Backend: Prisma
* Blockchain: Cardano
* Identity Layer: Atala PRISM
* Deployment: Vercel



## Getting Started

Follow the steps below to run CivicChain locally on your machine.



## Installation

* Ensure Node.js (v18 or higher) is installed
* Ensure npm or yarn is installed
* Ensure Git is installed

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/CivicChain.git
cd CivicChain
npm install
# or
yarn
```



## Running Locally

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open your browser and visit:

```text
http://localhost:3000
```


## Testing

If tests are configured, run:

```bash
npm test
# or
yarn test
```

Additional test configuration can be added depending on the testing framework used.



## Deployment

CivicChain is deployed using Vercel.

* Push the project to a GitHub repository
* Connect the repository to Vercel
* Configure required environment variables
* Deploy the application
